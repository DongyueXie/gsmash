Introduction

Try examples studied in Luke Smallman’s thesis.

sim_data1 = function(n,seed=12345){
  set.seed(seed)
  v1 = rpois(n,25)
  v2 = rpois(n,30)
  v3 = v1 + 3*v2
  lambda = cbind(v1,v1,v1,v1,v2,v2,v2,v2,v3,v3)
  Y = matrix(rpois(n*10,lambda),nrow=n)
  return(list(lambda=lambda,Y=Y))
}
datax=  sim_data1(100)
